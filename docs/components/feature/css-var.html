# 其他样式变量

组件库提供了一系列样式变量，用于快速搭建

# 卡片风格

  • 卡片默认圆角 --vi-card-radius
div {
    border-radius: var(--vi-card-radius)
}
  • 卡片阴影 --vi-card-box-shadow
div {
    box-shadow: var(--vi-card-box-shadow)
}
  • 卡片padding --vi-card-padding
div {
    padding: var(--vi-card-padding)
}
