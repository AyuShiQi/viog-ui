# 取色器 color-select new 待优化

用于选择颜色

# 使用说明

错误提示

组件目前无法精准解析16进制颜色，后续版本将优化

# 简单引入

通过绑定modelValue获取当前颜色，组件点击确定即可更改并获取颜色

#
展开代码
<template>
  <vi-color-select v-model="pick"/>
</template>
<script>
  const pick = ref('#aaaaaa')
</script>

# 显示十六进制

使用number属性显示当前颜色十六进制

#aaaaaa
#
展开代码
<template>
  <vi-color-select v-model="pick" number/>
</template>
<script>
  const pick = ref('#aaaaaa')
</script>

# 透明度控制

使用alpha属性开启透明度选择

#
展开代码
<template>
  <vi-color-select v-model="pick" alpha/>
</template>
<script>
  const pick = ref('#aaaaaa')
</script>
