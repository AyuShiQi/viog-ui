# 字体

组件并未规定字体风格，一般情况下展示为项目的默认字体，如有需要可以自行引用字体

# 字体大小

组件库规定了一系列的字体大小样式

h1 36px
h2 32px
h3 28px
h4 24px
h5 22px
h6 20px
p 14px

# 字体变量

组件库提供三种大小的一般字体，分别为：（--vi-big-font-size、--vi-font-size、--vi-small-font-size）

Viog UI
Viog UI
Viog UI

这三种字体对应组件中size属性控制的三种字体样式，你也可以直接使用这些字体变量
