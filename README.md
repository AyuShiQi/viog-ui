# viog-ui 0.2.4

https://ayushiqi.github.io/viog-ui/

基于vue3的炫酷ui组件库

特点：活力性  可玩性  原子性

> 组件库使用方式参考elementUI

## 使用

```
# npm
npm install viog-ui

#yarn
yarn add viog-ui
```

- 特殊标注样式属性请添加前缀--vi-组件名-属性名：如Scroll组件width：--vi-scroll-width

# 引入

``` javascript
import viog from 'viog-ui';
import from 'viog-ui/dist/viog-ui.css'

app.use(viog)
```
